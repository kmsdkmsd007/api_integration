This project showcases a clean and scalable approach to handling API integration, authentication, and session management in Flutter. It follows a structured clean-architecture pattern and includes complete login and signup flows with BLoC or Cubit state management.

## Features

* Modular architecture with clear separation of presentation, domain, and data layers
* Login and signup flows with remote and local data sources
* Repository pattern and use cases for business logic
* BLoC and Cubit for state management
* Dependency injection using a simple service locator
* Error and failure mapping using Either types
* Session handling with SharedPreferences
* Unit and bloc tests using mocktail and bloc_test

## Tech Stack

* **Framework:** Flutter, Dart
* **Architecture:** Clean Architecture
* **Networking:** Abstracted HTTP client
* **Local Storage:** SharedPreferences
* **Testing:** bloc_test, mocktail
